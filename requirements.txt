Instruções para Utilizar o Jupyter Notebook e as Bibliotecas

1. Instalar o Python
Certifique-se de ter o Python instalado no seu computador. Baixe a versão mais recente aqui:
https://www.python.org/downloads/

2. Instalar o Jupyter Notebook
Depois de instalar o Python, você pode instalar o Jupyter Notebook usando o pip:
pip install jupyterlab

Para abrir o Jupyter Notebook, execute no terminal:
jupyter notebook

Você verá uma interface web onde pode abrir os notebooks do projeto.

3. Instalar Bibliotecas do Projeto
Para instalar todas as dependências do projeto de uma vez:
    1. Navegue até o diretório do projeto onde está o arquivo requirements.txt.
    2. No terminal, execute o comando:
pip install -r requirements.txt

Isso instalará:
    • NumPy: Operações com arrays e álgebra linear.
    • Pandas: Manipulação e análise de dados.
    • Matplotlib: Criação de gráficos e visualizações.
    • Seaborn: Visualizações estatísticas mais avançadas.
    • SciPy: Computação científica.

5. Verificar a Instalação
Depois de instalar tudo, você pode verificar se as bibliotecas estão funcionando corretamente:
    1. Abra o Jupyter Notebook.
    2. Crie um novo notebook e insira o seguinte código para testar:

import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
import seaborn as sns
import scipy

print("Todas as bibliotecas foram instaladas com sucesso!")

Se tudo estiver correto, você verá a mensagem:
"Todas as bibliotecas foram instaladas com sucesso!"
